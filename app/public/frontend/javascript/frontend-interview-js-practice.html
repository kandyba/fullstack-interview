Методи масивів у JavaScript (reduce, filter, sort, map, forEach)

1. Сума масиву (reduce)

Завдання: Повернути суму всіх значень у масиві.

const array = [1, 2, 3, 4, 5];
const sumOfArray = array.reduce((a, b) => a + b, 0);
console.log(sumOfArray); // 15

Пояснення:

  • reduce() проходить по всьому масиву і накопичує результат.
  • a — акумулятор (поточна сума).
  • b — поточне значення елемента.
  • 0 — початкове значення.

Тип методу: акумулює значення в одне (наприклад, суму, середнє або конкатенацію рядків).

2. Фільтрація парних чисел (filter)

Завдання: Отримати лише парні числа з масиву.

const inputs = [1, 2, 3, 4, 5, 6];
const result = inputs.filter(x => x % 2 === 0);
console.log(result); // [2, 4, 6]

Пояснення: filter() створює новий масив, що містить лише елементи, які проходять перевірку умови.

У даному прикладі x % 2 === 0 залишає лише парні числа.

Тип методу: створює новий масив із відфільтрованими елементами.

3. Сортування парних чисел (sort)

Завдання: Відсортувати масив так, щоб парні числа йшли перед непарними.

const data = [5, 2, 8, 3, 1, 4];
data.sort((a, b) => a % 2 - b % 2);
console.log(data); // [2, 8, 4, 5, 3, 1]

Пояснення:

  • a % 2 повертає 0 для парних і 1 для непарних чисел.
  • Якщо a парне, а b непарне — результат буде від’ємним, тому a стане раніше.

Тип методу: змінює початковий масив, впорядковуючи елементи за заданою умовою.

4. Використання forEach, map, filter, reduce

Порівняння методів:

МетодПризначенняПовертає
forEach()Виконує дію над кожним елементомundefined
map()Створює новий масив із результатів функціїНовий масив
filter()Вибирає елементи, що відповідають умовіНовий масив
reduce()Перетворює масив у одне значенняБудь-яке значення

Приклад комбінації методів:

const arr = [1, 2, 3, 4, 5];
const result = arr
  .filter(x => x % 2 === 0) // [2, 4]
  .map(x => x * 2)          // [4, 8]
  .reduce((a, b) => a + b); // 12
console.log(result);

Пояснення: тут ми спочатку фільтруємо парні, потім подвоюємо їх і підсумовуємо результат.

5. Знайти мінімальне та максимальне значення в масиві

Завдання: Знайти min і max без циклу for.

const numbers = [3, 7, 1, 9, 2];
const min = Math.min(...numbers);
const max = Math.max(...numbers);
console.log(min, max); // 1 9

Пояснення: оператор ... розгортає масив у список аргументів.

6. Підрахунок кількості кожного елемента в масиві

Завдання: Порахувати, скільки разів зустрічається кожне слово.

const words = ['apple', 'banana', 'apple', 'orange', 'banana', 'apple'];
const count = words.reduce((acc, el) => {
  acc[el] = (acc[el] || 0) + 1;
  return acc;
}, {});
console.log(count); // { apple: 3, banana: 2, orange: 1 }

Тип методу: reduce() — створює об’єкт-лічильник.

7. Знайти унікальні значення

Завдання: Видалити дублікати з масиву.

const arr = [1, 2, 2, 3, 4, 4, 5];
const unique = [...new Set(arr)];
console.log(unique); // [1, 2, 3, 4, 5]

Пояснення: Set зберігає лише унікальні значення.

8. Перевірити, чи всі елементи парні (every)
const nums = [2, 4, 6, 8];
const allEven = nums.every(n => n % 2 === 0);
console.log(allEven); // true

Пояснення: every() повертає true, якщо всі елементи задовольняють умову.

9. Перевірити, чи є хоча б одне від’ємне число (some)
const numbers = [3, -2, 7, 4];
const hasNegative = numbers.some(n => n < 0);
console.log(hasNegative); // true

Пояснення: some() повертає true, якщо хоча б один елемент відповідає умові.

10. Знайти перший елемент, що відповідає умові (find)
const users = [
  { name: 'Alex', age: 20 },
  { name: 'John', age: 30 },
  { name: 'Bob', age: 25 }
];
const result = users.find(u => u.age > 25);
console.log(result); // { name: 'John', age: 30 }

Пояснення: find() повертає перший знайдений елемент або undefined.

11. Отримати масив тільки з імен користувачів (map)
const users = [
  { name: 'Alice', age: 25 },
  { name: 'Bob', age: 30 }
];
const names = users.map(u => u.name);
console.log(names); // ['Alice', 'Bob']

Тип методу: map() створює новий масив із перетворених значень.

12. Об’єднати кілька масивів у один (concat / flat)
const a = [1, 2];
const b = [3, 4];
const c = [5, 6];
const merged = a.concat(b, c);
console.log(merged); // [1, 2, 3, 4, 5, 6]

// або
const nested = [[1, 2], [3, 4]];
console.log(nested.flat()); // [1, 2, 3, 4]
13. Середнє арифметичне чисел у масиві
const nums = [10, 20, 30, 40];
const avg = nums.reduce((a, b) => a + b) / nums.length;
console.log(avg); // 25

Пояснення: reduce() підсумовує елементи, потім ділимо на кількість.

14. Отримати масив чисел з рядка
const input = "5,10,15,20";
const numbers = input.split(',').map(Number);
console.log(numbers); // [5, 10, 15, 20]

Пояснення: split() перетворює рядок у масив, map(Number) конвертує елементи в числа.

15. Згрупувати об’єкти за властивістю
const users = [
  { name: 'Alice', city: 'Kyiv' },
  { name: 'Bob', city: 'Lviv' },
  { name: 'Tom', city: 'Kyiv' }
];
const grouped = users.reduce((acc, user) => {
  (acc[user.city] = acc[user.city] || []).push(user);
  return acc;
}, {});
console.log(grouped);
// { Kyiv: [{...}, {...}], Lviv: [{...}] }

Пояснення: типове завдання на reduce() — групування за ключем.

16. Підрахувати кількість ключів в об’єкті
const user = { name: 'Alex', age: 25, city: 'Kyiv' };
const count = Object.keys(user).length;
console.log(count); // 3

Пояснення: Object.keys() повертає масив усіх ключів об’єкта, і через .length дізнаємось їхню кількість.

17. Злиття двох об’єктів
const a = { name: 'Alice', age: 25 };
const b = { city: 'Lviv', age: 30 };
const merged = { ...a, ...b };
console.log(merged); // { name: 'Alice', age: 30, city: 'Lviv' }

Пояснення: деструктуризація дозволяє об’єднувати об’єкти, а пізніші ключі перекривають попередні.

18. Перевірити, чи існує властивість у об’єкті
const user = { name: 'Bob', age: 30 };
console.log('age' in user); // true
console.log(user.hasOwnProperty('email')); // false

Пояснення: in перевіряє наявність ключа навіть у прототипі, а hasOwnProperty() — лише у власних властивостях.

19. Глибоке копіювання об’єкта
const user = { name: 'Anna', address: { city: 'Kyiv' } };
const copy = structuredClone(user);
copy.address.city = 'Lviv';
console.log(user.address.city); // Kyiv

Пояснення: structuredClone() створює повну копію без спільних посилань. Альтернатива — JSON.parse(JSON.stringify(obj)).

20. Перетворити об’єкт на масив пар [ключ, значення]
const obj = { a: 1, b: 2, c: 3 };
const entries = Object.entries(obj);
console.log(entries); // [['a',1], ['b',2], ['c',3]]

Пояснення: Object.entries() — зручно для циклів, map або побудови таблиць.

21. Перетворити масив пар у об’єкт
const pairs = [['a', 1], ['b', 2], ['c', 3]];
const obj = Object.fromEntries(pairs);
console.log(obj); // { a: 1, b: 2, c: 3 }

Типове завдання: обернене до Object.entries().

22. Знайти ключ за значенням
const colors = { red: '#f00', green: '#0f0', blue: '#00f' };
const key = Object.keys(colors).find(k => colors[k] === '#0f0');
console.log(key); // 'green'

Пояснення: Object.keys() + find() дозволяють швидко знайти ключ по значенню.

23. Згрупувати об’єкти за ключем
const users = [
  { name: 'Alex', city: 'Kyiv' },
  { name: 'John', city: 'Lviv' },
  { name: 'Kate', city: 'Kyiv' }
];
const grouped = users.reduce((acc, u) => {
  (acc[u.city] = acc[u.city] || []).push(u);
  return acc;
}, {});
console.log(grouped);
// { Kyiv: [{Alex}, {Kate}], Lviv: [{John}] }

Пояснення: класична задача на reduce() — групування по властивості.

24. Отримати вкладену властивість без помилки (Optional chaining)
const data = { user: { profile: { name: 'Olena' } } };
const name = data?.user?.profile?.name || 'Unknown';
console.log(name); // 'Olena'

Пояснення: оператор ?. запобігає помилці, якщо властивість відсутня.

25. Сортування масиву об’єктів за ключем
const users = [
  { name: 'Bob', age: 25 },
  { name: 'Alice', age: 30 },
  { name: 'Tom', age: 20 }
];
users.sort((a, b) => a.age - b.age);
console.log(users);
// [{Tom,20}, {Bob,25}, {Alice,30}]

Пояснення: sort() змінює оригінальний масив. Дуже типове завдання “відсортуй об’єкти за віком”.

26. Перевернути рядок (reverse string)
const str = 'JavaScript';
const reversed = str.split('').reverse().join('');
console.log(reversed); // 'tpircSavaJ'

Пояснення: split('') перетворює рядок у масив символів, reverse() змінює порядок, join('') знову збирає в рядок.

27. Перевірити, чи є рядок паліндромом
const str = 'level';
const isPalindrome = str === str.split('').reverse().join('');
console.log(isPalindrome); // true

Пояснення: паліндром читається однаково з обох боків.

28. Зробити першу букву великою (capitalize)

const capitalized = str.charAt(0).toUpperCase() + str.slice(1);
console.log(capitalized); // 'Hello world'

Пояснення: charAt(0) бере першу букву, slice(1) — решту рядка.

29. Зробити великою першу букву кожного слова

Пояснення: часто дають для перевірки split + map.

30. Порахувати кількість голосних у рядку

37. Факторіал числа (рекурсія)
function factorial(n) {
  return n === 0 ? 1 : n * factorial(n - 1);
}
console.log(factorial(5)); // 120

Пояснення: рекурсивний виклик множить n на результат для n − 1.

38. Числа Фібоначчі (ітеративно)
function fibonacci(n) {
  const seq = [0, 1];
  for (let i = 2; i < n; i++) {
    seq[i] = seq[i - 1] + seq[i - 2];
  }
  return seq;
}
console.log(fibonacci(8)); // [0,1,1,2,3,5,8,13]
39. Знайти мінімум і максимум у масиві (без Math.min)

  if (num < min) min = num;
  if (num > max) max = num;
}

40. Перевірити, чи є два масиви анаграмами

         b.toLowerCase().split('').sort().join('');
}
console.log(isAnagram('listen', 'silent')); // true

Пояснення: анаграма — коли однаковий набір літер у різному порядку.

67. Реалізувати throttle для подій resize
function throttle(fn, delay) {
  let last = 0;
  return (...args) => {
    const now = Date.now();
    if (now - last > delay) {
      last = now;
      fn(...args);
    }
  };
}

window.addEventListener('resize', throttle(() => {
  console.log('Resize event (throttled)');
}, 500));

Пояснення: throttle обмежує частоту виконання функції.

75. Реалізувати memoization (кешування результатів)
function memoize(fn) {
  const cache = {};
  return (...args) => {
    const key = JSON.stringify(args);
    if (cache[key]) return cache[key];
    const result = fn(...args);
    cache[key] = result;
    return result;
  };
}

const slowSquare = n => { for(let i=0;i<1e7;i++); return n*n; };
const fastSquare = memoize(slowSquare);

console.time('1'); fastSquare(5); console.timeEnd('1');
console.time('2'); fastSquare(5); console.timeEnd('2');

Пояснення: повторні виклики з тими самими аргументами повертають результат миттєво.

77. Реалізувати debounce із leading edge
function debounce(fn, delay, immediate = false) {
  let timeout;
  return function(...args) {
    const callNow = immediate && !timeout;
    clearTimeout(timeout);
    timeout = setTimeout(() => timeout = null, delay);
    if (callNow) fn(...args);
  };
}

const log = debounce(() => console.log('Run'), 1000, true);
window.addEventListener('scroll', log);

Пояснення: immediate дозволяє викликати функцію одразу на першій події.

  emit(event, data) { (this.events[event] || []).forEach(fn => fn(data)); }
}
