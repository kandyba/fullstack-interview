Що таке Angular і які його основні переваги?

Angular - це платформа та фреймворк для створення одно-сторінкових клієнтських додатків з використанням HTML і TypeScript.

Основні переваги:

  • TypeScript - сильна типізація з коробки
  • Dependency Injection - вбудована система DI
  • CLI - потужний інструмент для генерації коду
  • RxJS - реактивне програмування
  • Two-way binding - двостороння прив'язка даних
Що таке компонент в Angular?

Компонент - це основний будівельний блок Angular додатку, який складається з:

  • @Component декоратора з метаданими
  • TypeScript класу з логікою
  • HTML шаблону
  • CSS стилів (optional)
@Component({
  selector: 'app-root',
  templateUrl: './app.component.html',
  styleUrls: ['./app.component.css']
})
export class AppComponent {
  title = 'My App';
}
Що таке директиви в Angular?

Директиви - це класи, які додають поведінку до елементів DOM.

Типи директив:

  • Structural - змінюють структуру DOM (*ngIf, *ngFor)
  • Attribute - змінюють вигляд або поведінку (ngClass, ngStyle)
  • Component - директиви з шаблоном
Що таке сервіси в Angular?

Сервіси - це класи, які містять бізнес-логіку та можуть бути впроваджені в компоненти через Dependency Injection.

@Injectable({
  providedIn: 'root'
})
export class DataService {
  getData() {
    return this.http.get('/api/data');
  }
}
Що таке lifecycle hooks в Angular?

Lifecycle hooks - це методи, які дозволяють реагувати на різні події життєвого циклу компонента:

  • ngOnInit() - ініціалізація після створення
  • ngOnChanges() - зміна input властивостей
  • ngOnDestroy() - перед знищенням компонента
  • ngAfterViewInit() - після ініціалізації view
  • ngDoCheck() - кастомна перевірка змін
